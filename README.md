### 🔭 What I'm working on

- **[Sweeten](https://sweeten-3aa2.vercel.app)** — AI-integrated diabetes management platform. Hybrid architecture: deterministic ADA-guideline safety layer + Gemini-based personalization. Built after identifying and correcting a critical flaw in early plan logic following consultation with practitioners at the Primary Care Diabetes Association (PCDA).
- **Oneiro / Clotho** — ongoing contributor to a multi-agent AI system controlling an autonomous Minecraft agent. Built the MCP bridge layer (TypeScript, MCP SDK, stdio transport) connecting the planner and dashboard to the agent's action layer.
- **Gender Bias in Profession Classification** — independent research project. Counterfactual evaluation of a DistilBERT profession classifier using the Bias in Bios dataset — measuring flip rate and confidence shift across 28 professions to isolate gender-cue sensitivity from content. Currently extending to a cross-model comparison (DistilBERT / BERT / RoBERTa).
